Quantification of Performance

ARCoach gives the users feedback about the exercise through hints and a performance score. The judgment of the performance score should be related to the professional coaches’ corresponding exercise. ARCoach borrows the idea from MSE (mean square error). Take jumping jacks as an example. ARCoach judges the performance by MSE value, more close MSE is to 0, more accurate the exercise. Squats Detection

It can be concluded that the position between the user and iPhone would affect the joint data of Squats from the below figure. Considered that part of the joint data is heavily impacted by the position, the algorithm should choose the joint data which has a clear “wave pattern” to determine a valid exercise count accurately. ARCoach chooses the hand joints and foot joints on axis-Y to detect a valid count, plus hand joints and foot joints on axis-Z to calculate the performance score of Squat workout.

Database

ARCoach uses Firebase as authentication and database. User registers in ARCoach and ARCoach would send the user’s email and password to Firebase and get a complicated UID in return. The user’s information would be uploaded into Firebase realtime database under the root of the user’s UID. The free version of Firebase supports up to 100 connections simultaneously. About ARCoach

This project aims to develop an iOS application ARCoach for giving instant fitness instructions and feedbacks armed with body motion tracking through the iPhone’s camera, providing exercises with virtual objects using immersive technology Augmented Reality(AR). It helps users to do exercises correctly at home without hiring a coach or buying extra wearable devices, meanwhile enjoy the immersive experience brought by AR. Unlike some fitness applications that need input data from a wearable device, ARCoach uses the camera of iPhone to track body motion and extract movement information, users, therefore, need not buy an extra wearable device for fitness.
